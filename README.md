# 🛼 LP Patins Animada

Landing page animada para loja de patins, com CSS animations, galeria de imagens e layout visual.

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Glossary/HTML5)
[![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/CSS)

---

## 🚀 Tecnologias

| Tecnologia | Função |
|---|---|
| **HTML5** | Estrutura |
| **CSS3** | Animações, layout responsivo e estilização |

## ✨ Funcionalidades

- ✅ Animações CSS (keyframes, transitions)
- ✅ Galeria de imagens
- ✅ CSS modular por seção (banner, gallery, header, footer)
- ✅ Layout responsivo

## 🏗️ Estrutura

```
/styles
├── global.css      # Reset
├── index.css       # Importações
├── banner.css      # Banner principal
├── gallery.css     # Galeria de imagens
├── header.css      # Navegação
├── footer.css      # Rodapé
├── hero.css        # Seção hero
```

## 🚦 Como Rodar

Abra `index.html` no navegador.

## 💡 O que foi aprendido

- Animações CSS com `@keyframes`
- Transições suaves entre estados
- Layout visual com imagens em grid
